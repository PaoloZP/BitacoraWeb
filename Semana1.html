Semana 1

La primera semana sirvió para conocernos, el profesor se presentó, nos contó un poco sobre él, a que se dedica, sus logros, sus hobbies y compartió también una que otra situación personal. Después, hizo que cada alumno se presente, diga un poco sobre sí mismo, la carrera que está estudiando y alguna cualidad destacada.

Luego de presentarnos, nos dio una introducción sobre el curso; de qué constaba, sus métodos de calificación, las tardanzas, los permisos, etc. Así concluyo la primera clase presencial.

En la clase virtual, tuvimos que descargarnos Visual Studio Code y crear una cuenta en Github para así empezar a crear nuestra Bitácora digital. Practicamos un poco creando la frase “hola mundo” en Index.html. Luego, se nos dio una breve introducción sobre como crear una web usando plantillas de HTML, lo que solo haría que nosotros la personalicemos para nuestra Bitácora digital de acuerdo con nuestro gusto, así al final podamos compartir el resultado mediante un link.
